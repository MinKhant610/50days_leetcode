Question link :https://leetcode.com/problems/merge-sorted-array/

Solution :

class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int], n: int) -> None:
        """
        Do not return anything, modify nums1 in-place instead.
        """

        first = nums1[:m]
        second = nums2[:n]

        result = sorted(first + second)
        nums1[:] = result
